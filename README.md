## Folder and file structure
```
./
├── README.md
├── LICENSE.md
├── index.html                                 * home page
├── favicon.ico
|
├── screenshots/                               * responsive test screenshots
|
├── static/                                    * site source
│   ├── images/                                * image sources
|   |   └── bg/                                * background images
|   │
│   └── css/                                   * style scripts
│
├── pages/                                     * rest of pages
├── docs/                                      * info docs
└── wallet/                                    * Bitcoin-express wallet files
```
